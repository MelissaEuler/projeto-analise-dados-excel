# Projeto de Análise de Dados com Excel

## Descrição
Este projeto foi desenvolvido com o objetivo de aplicar conceitos de análise de dados utilizando o Microsoft Excel.  
A partir de uma base de dados fornecida, foram criadas tabelas dinâmicas, gráficos e um dashboard para facilitar a visualização e interpretação das informações.

## Ferramentas Utilizadas
- Microsoft Excel
- Tabelas Dinâmicas
- Gráficos
- Dashboard

## Análises Realizadas

### Faturamento por Plano
Analisa o total de receita gerada por cada plano, permitindo identificar quais planos são os principais responsáveis pelo faturamento do negócio.

### Quantidade de Assinantes por Plano
Apresenta o volume de clientes em cada plano, auxiliando na compreensão da distribuição da base de usuários e do comportamento de adesão aos planos.

### Ticket Médio por Plano
Calcula o valor médio gerado por assinante em cada plano, permitindo avaliar o valor individual de cada cliente e identificar oportunidades de crescimento por meio de upgrades.

## Dashboard
O dashboard final consolida os principais indicadores em uma visão única, facilitando a análise comparativa entre os planos e apoiando a tomada de decisão baseada em dados.

## Objetivo do Projeto
Consolidar o aprendizado sobre análise de dados, organização de informações e criação de dashboards no Excel.
